MIS Esquipulas: Caja

Bienvenido al sistema de ayuda de la aplicación de Caja en MIS Esquipulas

El modulo de caja fue diseñado para almacenar todas las transacciones de ingresos y egresos a diario, para iniciar el proceso se debe cargar la caja con un monto inicial X para poder realizar egresos en cualquier momento a pagos en efectivo requeridos.

En este modulo las transacciones se llevan a cabo a partir de los siguientes movimientos:

  1. "Movimientos"
    1. Apertura de Caja
    2. Facturas
    3. Recibos
    4. Pagos en Efectivo
    5. Devoluciones
    6. Arqueo
