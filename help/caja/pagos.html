MIS Esquipulas Caja: Pagos en Efectivo

Este proceso se lleva a cabo cuando se necesitan realizar pagos inmediatos y urgentes. El encargado realiza un documento con este modulo bajo algun concepto, para constatar la salida y entrega del dinero, sin embargo este proceso requiere de una previa autorización por uno de los siguientes: gerente, contador, financiero o presidente. Esta autorización es el documento que soporta la transacción.
