MIS Esquipulas Caja: Devoluciones

Este proceso es realizado por el cajero, se lleva a cabo cuando los clientes hacen devoluciones totales o parciales de las mercancías, para su contabilización se utiliza un comprobante llamado nota crédito, documento que es creado en este formulario. Este comprobante se prepara en original y dos copias; el original para el cliente, una copia para el archivo consecutivo y otra para anexar al comprobante diario de contabilidad.

Este proceso esta directamente vinculado con el modulo devoluciones en Inventario, en el modulo de inventario se actualiza la existencia de productos y en caja, se realiza el documento correspondiente para contabilizar la devolución de mercadería, debiendo ser realizada primeramente la nota de credito en caja para poder actualizar las existencias.
