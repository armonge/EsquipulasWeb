Creando una nueva Entrara Local

Para crear una nueva entrada local se utiliza el icono de nuevo (  ) y se mostrara entonces el formulario en modo de edición para que usted pueda introducir todos los datos pertinentes, para poder guardar la entrada es necesario  que se especifique el proveedor, la fecha, el tipo de pago y por lo menos un producto con su cantidad y precio de compra
