MIS Esquipulas: Inventario

Bienvenido al sistema de ayuda de la aplicación de Inventario en MIS Esquipulas

El sistema de control de inventario fue diseñado para manejar los productos en existencia, registrar las entradas por compras y salidas por ventas o cualquier otro concepto. Los catalogos de productos, movimientos de inventarios y reportes fueron creados según las especificaciones solicitadas por la organización. La información de esta aplicación reside en una base de datos de MySql Server y los reportes son accesibles via Web desde internet, brindando la posibilidad de distruibuir la informaci´n de la forma mas rapida posible para mejorar el apoyo a la toma de decisiones.

En este modulo podra encontrar lo siguiente:

  1. "Movimientos"
    1. Liquidaciones
      2. Modulo para Liquidar costos de los Articulos importados e ingreso a Inventario.
    2. Entradas Locales
      3. Modulo para ingresar a inventario las compras a nivel nacional.
  2. Kardex
    1. "Entrada Locales, Importaciones y Devoluciones (Entrada)"
      2. Modulo para dar entrada a la existencia de articulos en inventario (Liquidacion de costos, Compra Local o Devolucion)
    2. Salidas por Factura
      3. Modulo para dar salida a Articulos del inventario cuando se realiza una venta.
    3. Kardex por otros conceptos
      4. Modulo para dar salida o entrada a Articulos del inventario bajo otros conceptos.
  3. Catálogos
    1. Catalogo de articulos
      2. Presenta pantalla de manejo las l&aicute;neas de productos
    2. Catalogo de categorias
      3. Presenta pantalla de manejo de inventario
    3. "Catalogo de Marcas"
      4. Permite Navegar y agregar marcas al sistema.
    4. Catalogo de proveedores
