MIS Esquipulas: Inventario

Bienvenido al sistema de ayuda de la aplicación de Inventario en MIS Esquipulas

El modulo de control de inventario fue diseñado para manejar los productos en existencia, registrar las entradas por compras y salidas por ventas o cualquier otro concepto. Los catalogos de productos, movimientos de inventarios y reportes fueron creados según las especificaciones solicitadas por la organización. Los reportes de este sistema son accesibles via Web desde internet.

En este modulo podra encontrar lo siguiente:

  1. Movimientos
    1. Liquidaciones Modulo para Liquidar costos de los Articulos importados e ingreso a Inventario.
    2. Entradas Locales Modulo para ingresar a inventario las compras a nivel nacional.
  2. Kardex
    1. Entradas Locales, Importaciones y Devoluciones (Entrada) Modulo para dar entrada a la existencia de articulos en inventario (Liquidacion de costos, Compra Local o Devolucion)
    2. Salidas por Factura Modulo para dar salida a Articulos del inventario cuando se realiza una venta.
    3. Kardex por otros conceptos Modulo para dar salida o entrada a Articulos del inventario bajo otros conceptos.
  3. Catálogos
    1. Catalogo de articulos Presenta pantalla de manejo las láneas de productos
    2. Catalogo de categorias Presenta pantalla de manejo de inventario
    3. Catalogo de Marcas Permite Navegar y agregar marcas al sistema.
    4. Catalogo de proveedores
